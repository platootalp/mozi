# Config Module (配置模块)

## 文档信息

| 字段 | 内容 |
|------|------|
| 模块名称 | Config |
| 职责 | 配置存储、错误码、降级策略 |
| 路径 | `mozi/config/` |
| 文档版本 | v1.0 |
| 状态 | 规划中 |
| 创建日期 | 2026-03-28 |

---

## 1. 配置项

| 配置项 | 来源 | 说明 |
|--------|------|------|
| `MOZI_MODEL` | 环境变量 | 默认模型 |
| `MOZI_CONFIG_PATH` | 环境变量 | 配置文件路径 |
| `~/.mozi/config.json` | 配置文件 | 用户配置 |

## 2. 错误码

| 错误码 | 说明 |
|--------|------|
| E1001 | 模型调用失败 |
| E1002 | 工具执行失败 |
| E2001 | 会话不存在 |
| E3001 | 权限不足 |
| E4001 | 配置错误 |

## 3. 降级策略

| 场景 | 降级行为 |
|------|----------|
| 模型超时 | 重试 3 次，失败后返回错误 |
| 工具执行失败 | 返回错误信息给用户 |
| 存储服务不可用 | 降级到内存模式 |

## 4. 弹性模式

- **重试机制**：模型调用、工具执行支持指数退避重试
- **熔断器**：连续失败 5 次后熔断 60s
- **超时控制**：每个工具调用超时 30s

## 5. 依赖关系

- **依赖模块**: 无
- **被依赖模块**: 所有模块

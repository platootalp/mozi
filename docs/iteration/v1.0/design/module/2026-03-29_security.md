# Security 模块设计文档

## 1. 模块概述

### 1.1 模块名称

Security（安全模块）

### 1.2 职责

Security 模块是 Mozi AI Coding Agent 的横切面安全组件，负责：
- 提供统一的安全检查机制
- 管理工具执行白名单
- 实现沙箱隔离保护
- 检测和阻止危险函数调用
- 管理密钥和凭证的安全访问

### 1.3 核心能力

| 能力 | 说明 |
| ---- | ---- |
| 权限模型 | 基于角色的访问控制，细粒度权限配置 |
| 工具白名单 | 支持通配符的路径和命令白名单验证 |
| 沙箱隔离 | 进程级隔离，限制危险操作范围 |
| 危险函数检测 | AST 静态分析检测黑名单函数 |
| 密钥管理 | 环境变量注入，禁止硬编码检测 |
| 安全审计 | 操作日志记录，安全事件追溯 |

### 1.4 核心问题

| 问题 | 说明 |
| ---- | ---- |
| 恶意代码执行 | 用户输入可能包含危险命令或代码 |
| 路径遍历 | 文件操作可能访问白名单外的路径 |
| 敏感信息泄露 | 密钥、密码等敏感信息被意外暴露 |
| 权限失控 | 工具执行权限过大，缺乏细粒度控制 |

### 1.5 难点

| 难点 | 说明 |
| ---- | ---- |
| 危险函数检测 | eval/exec 等函数如何准确检测，避免误报 |
| 沙箱隔离 | 如何在限制权限的同时保持功能可用 |
| 白名单设计 | 如何设计既安全又方便的白名单规则 |
| AST 分析性能 | 静态代码分析对大型文件的性能影响 |

### 1.6 解决方案

| 方案 | 说明 |
| ---- | ---- |
| DangerousFunctionDetector | AST 静态分析，检测黑名单函数调用 |
| SandboxExecutor | 进程级隔离，限制文件和网络访问 |
| WhitelistManager | 通配符支持的白名单验证机制 |
| SecretManager | 密钥通过环境变量注入，扫描禁止硬编码 |

## 2. 安全模型

### 2.1 权限体系

Mozi 采用基于上下文的权限模型，权限信息通过 `SecurityContext` 传递：

```
SecurityContext
    ├── user_id: 用户标识
    ├── session_id: 会话标识
    ├── permissions: 权限列表
    ├── working_directory: 工作目录范围
    ├── allowed_tools: 允许执行的工具列表
    ├── allowed_paths: 允许访问的路径白名单
    └── resource_limits: 资源限制（超时/内存等）
```

### 2.2 访问控制流程

```
工具执行请求
    │
    ▼
SecurityContext 构建
    │
    ▼
权限检查（permission check）
    │
    ├──► 无权限 ──► SecurityViolationError
    │
    ▼
白名单验证（whitelist validation）
    │
    ├──► 不在白名单 ──► SecurityViolationError
    │
    ▼
沙箱隔离检查（sandbox check）
    │
    ├──► 违反沙箱规则 ──► SecurityViolationError
    │
    ▼
危险函数检测（dangerous function scan）
    │
    ├──► 检测到危险函数 ──► SecurityViolationError
    │
    ▼
执行工具
```

### 2.3 权限级别

| 级别 | 名称 | 描述 |
| ---- | ---- | ---- |
| 0 | NONE | 无任何权限 |
| 1 | READ | 仅允许读取文件 |
| 2 | WRITE | 允许读写文件 |
| 3 | EXECUTE | 允许执行命令 |
| 4 | ADMIN | 管理员权限 |

## 3. 工具白名单

### 3.1 白名单配置

白名单支持三种匹配模式：

| 模式 | 示例 | 说明 |
| ---- | ---- | ---- |
| 精确匹配 | `/Users/lijunyi/road/mozi` | 精确路径 |
| 通配符匹配 | `/Users/lijunyi/road/mozi/**` | 匹配目录下所有文件 |
| 正则匹配 | `^/tmp/.*\.tmp$` | 正则表达式匹配 |

### 3.2 白名单管理接口

```python
class WhitelistManager:
    """白名单管理器"""

    def __init__(self) -> None:
        self._path_whitelist: list[str] = []
        self._command_whitelist: list[str] = []

    def add_path(self, pattern: str) -> None:
        """添加路径白名单"""
        self._validate_pattern(pattern)
        self._path_whitelist.append(pattern)

    def add_command(self, pattern: str) -> None:
        """添加命令白名单"""
        self._validate_pattern(pattern)
        self._command_whitelist.append(pattern)

    def is_path_allowed(self, path: str) -> bool:
        """检查路径是否在白名单内"""
        resolved = self._resolve_path(path)
        return self._match_any(resolved, self._path_whitelist)

    def is_command_allowed(self, command: str) -> bool:
        """检查命令是否在白名单内"""
        return self._match_any(command, self._command_whitelist)

    def _match_any(self, value: str, patterns: list[str]) -> bool:
        """匹配任意白名单模式"""
        for pattern in patterns:
            if self._matches(value, pattern):
                return True
        return False

    def _matches(self, value: str, pattern: str) -> bool:
        """单个模式匹配"""
        if pattern.endswith("/**"):
            prefix = pattern[:-3]
            return value.startswith(prefix)
        elif "**" in pattern:
            import re
            regex = self._wildcard_to_regex(pattern)
            return bool(re.match(regex, value))
        else:
            return value == pattern

    def _wildcard_to_regex(self, pattern: str) -> str:
        """将通配符转换为正则表达式"""
        import re
        regex = re.escape(pattern)
        regex = regex.replace(r"\*\*", ".*")
        regex = regex.replace(r"\*", "[^/]*")
        return f"^{regex}$"
```

## 4. 沙箱隔离

### 4.1 沙箱策略

Mozi 采用多层次沙箱隔离策略：

| 层级 | 隔离方式 | 保护目标 |
| ---- | -------- | -------- |
| 进程级 | subprocess + 列表参数 | 防止命令注入 |
| 文件系统 | chroot/jail + 路径白名单 | 防止路径遍历 |
| 网络 | 禁用网络访问 | 防止数据外泄 |
| 资源 | cgroup + ulimit | 防止资源耗尽 |

### 4.2 进程沙箱实现

```python
class SandboxExecutor:
    """沙箱执行器"""

    def __init__(
        self,
        timeout: int = 30,
        memory_limit_mb: int = 512,
        allowed_paths: list[str] | None = None,
    ) -> None:
        self._timeout = timeout
        self._memory_limit_mb = memory_limit_mb
        self._allowed_paths = allowed_paths or []

    async def execute_command(
        self,
        command: list[str],
        working_directory: str,
    ) -> CommandResult:
        """在沙箱中执行命令"""
        # 验证工作目录
        if not self._is_path_allowed(working_directory):
            raise SecurityViolationError(
                f"Working directory not allowed: {working_directory}"
            )

        # 验证所有命令参数
        for arg in command:
            if not self._is_safe_argument(arg):
                raise SecurityViolationError(
                    f"Potentially dangerous argument: {arg}"
                )

        # 构建进程环境
        env = self._build_safe_env()

        # 执行命令（禁止 shell=True）
        process = await asyncio.create_subprocess_exec(
            *command,
            stdout=asyncio.subprocess.PIPE,
            stderr=asyncio.subprocess.PIPE,
            cwd=working_directory,
            env=env,
        )

        try:
            stdout, stderr = await asyncio.wait_for(
                process.communicate(),
                timeout=self._timeout,
            )
            return CommandResult(
                success=process.returncode == 0,
                stdout=stdout.decode(),
                stderr=stderr.decode(),
                exit_code=process.returncode,
            )
        except asyncio.TimeoutError:
            process.kill()
            raise SecurityViolationError(
                f"Command timed out after {self._timeout} seconds"
            )

    def _is_safe_argument(self, arg: str) -> bool:
        """检查命令参数是否安全"""
        dangerous_patterns = [
            ";", "&", "|", "`", "$(", "${",
            ">", "<", "\n", "\r",
        ]
        return not any(pattern in arg for pattern in dangerous_patterns)
```

### 4.3 路径遍历防护

```python
class PathSecurity:
    """路径安全检查"""

    @staticmethod
    def validate_and_resolve(path: str, base: str) -> str:
        """验证并解析路径，防止路径遍历"""
        import os
        from pathlib import Path

        base_path = Path(base).resolve()
        target_path = (base_path / path).resolve()

        # 检查解析后的路径是否在 base 目录内
        try:
            target_path.relative_to(base_path)
        except ValueError:
            raise SecurityViolationError(
                f"Path traversal attempt detected: {path}"
            )

        return str(target_path)

    @staticmethod
    def is_safe_path(path: str) -> bool:
        """检查路径是否安全（无遍历风险）"""
        normalized = os.path.normpath(path)
        return not (
            normalized.startswith("..") or
            normalized.startswith("/etc") or
            normalized.startswith("/root") or
            normalized.startswith("/sys")
        )
```

## 5. 危险函数检测

### 5.1 黑名单函数

| 函数 | 风险等级 | 替代方案 |
| ---- | -------- | -------- |
| eval() | HIGH | ast.literal_eval() |
| exec() | HIGH | 特定功能函数 |
| pickle.loads() | HIGH | json.loads() |
| subprocess.shell=True | HIGH | 列表参数传递 |
| __import__() | MEDIUM | 静态导入 |
| compile() | MEDIUM | ast.parse() |
| open() | LOW | 路径白名单验证后使用 |

### 5.2 AST 静态分析检测器

```python
import ast
from dataclasses import dataclass


@dataclass
class DangerousFunction:
    name: str
    lineno: int
    col_offset: int
    severity: str
    message: str


class DangerousFunctionDetector(ast.NodeVisitor):
    """危险函数 AST 检测器"""

    DANGEROUS_FUNCTIONS: dict[str, tuple[str, str]] = {
        "eval": ("HIGH", "eval() allows arbitrary code execution"),
        "exec": ("HIGH", "exec() allows arbitrary code execution"),
        "compile": ("MEDIUM", "compile() can create executable code"),
        "__import__": ("MEDIUM", "__import__() has dynamic import risk"),
        "load": ("HIGH", "pickle.load() has deserialization vulnerability"),
        "loads": ("HIGH", "pickle.loads() has deserialization vulnerability"),
    }

    def __init__(self) -> None:
        self._violations: list[DangerousFunction] = []

    def visit_Call(self, node: ast.Call) -> None:
        """检测危险函数调用"""
        if isinstance(node.func, ast.Name):
            func_name = node.func.id
            if func_name in self.DANGEROUS_FUNCTIONS:
                severity, message = self.DANGEROUS_FUNCTIONS[func_name]
                self._violations.append(DangerousFunction(
                    name=func_name,
                    lineno=node.lineno or 0,
                    col_offset=node.col_offset or 0,
                    severity=severity,
                    message=message,
                ))
        self.generic_visit(node)

    def get_violations(self) -> list[DangerousFunction]:
        """获取检测到的违规列表"""
        return self._violations.copy()


class DangerousFunctionScanner:
    """危险函数扫描器"""

    def scan_file(self, file_path: str) -> list[DangerousFunction]:
        """扫描文件检测危险函数"""
        with open(file_path, "r", encoding="utf-8") as f:
            source = f.read()

        tree = ast.parse(source, filename=file_path)
        detector = DangerousFunctionDetector()
        detector.visit(tree)

        return detector.get_violations()

    def scan_code(self, code: str) -> list[DangerousFunction]:
        """扫描代码字符串检测危险函数"""
        tree = ast.parse(code)
        detector = DangerousFunctionDetector()
        detector.visit(tree)

        return detector.get_violations()

    def check_and_raise(self, code: str) -> None:
        """检查并抛出异常"""
        violations = self.scan_code(code)
        if violations:
            messages = "\n".join(
                f"  Line {v.lineno}: {v.name} - {v.message}"
                for v in violations
            )
            raise SecurityViolationError(
                f"Dangerous functions detected:\n{messages}"
            )
```

## 6. 密钥管理

### 6.1 密钥管理原则

| 原则 | 说明 |
| ---- | ---- |
| 禁止硬编码 | 密钥必须通过环境变量或密钥服务注入 |
| 最小暴露 | 密钥只在必要时传递，使用后立即清理 |
| 审计追溯 | 所有密钥访问必须记录日志 |

### 6.2 环境变量注入接口

```python
class SecretManager:
    """密钥管理器"""

    def __init__(self) -> None:
        self._secrets: dict[str, str] = {}
        self._access_log: list[SecretAccessLog] = []

    def load_from_environment(self, *keys: str) -> None:
        """从环境变量加载密钥"""
        import os
        for key in keys:
            value = os.environ.get(key)
            if value is not None:
                self._secrets[key] = value

    def get(self, key: str) -> str | None:
        """获取密钥值"""
        self._log_access(key)
        return self._secrets.get(key)

    def get_required(self, key: str) -> str:
        """获取必需的密钥，不存在则抛异常"""
        value = self.get(key)
        if value is None:
            raise KeyError(f"Required secret not found: {key}")
        return value

    def inject_into_context(self, context: dict[str, Any]) -> None:
        """将密钥注入到上下文（仅注入非敏感字段）"""
        safe_keys = {"api_key", "api_key", "access_token"}
        for key, value in self._secrets.items():
            if any(safe in key.lower() for safe in safe_keys):
                context[key] = value

    def clear(self) -> None:
        """清除所有密钥内存"""
        self._secrets.clear()

    def _log_access(self, key: str) -> None:
        """记录密钥访问日志"""
        import datetime
        self._access_log.append(SecretAccessLog(
            key=key,
            accessed_at=datetime.datetime.utcnow(),
        ))
```

### 7.1 核心类型定义

```python
from __future__ import annotations

from abc import ABC, abstractmethod
from dataclasses import dataclass, field
from datetime import datetime
from enum import IntEnum
from typing import Any


class PermissionLevel(IntEnum):
    """权限级别枚举"""
    NONE = 0
    READ = 1
    WRITE = 2
    EXECUTE = 3
    ADMIN = 4


class ViolationSeverity(IntEnum):
    """违规严重级别"""
    LOW = 1
    MEDIUM = 2
    HIGH = 3
    CRITICAL = 4


@dataclass
class SecurityContext:
    """安全上下文"""
    user_id: str = "default"
    session_id: str | None = None
    permissions: list[PermissionLevel] = field(default_factory=list)
    working_directory: str = "."
    allowed_tools: list[str] = field(default_factory=list)
    allowed_paths: list[str] = field(default_factory=list)
    resource_limits: dict[str, Any] = field(default_factory=dict)

    def has_permission(self, level: PermissionLevel) -> bool:
        """检查是否具有指定权限级别"""
        if PermissionLevel.ADMIN in self.permissions:
            return True
        return level in self.permissions


@dataclass
class SecurityViolationError:
    """安全违规错误"""
    message: str
    severity: ViolationSeverity = ViolationSeverity.HIGH
    details: dict[str, Any] = field(default_factory=dict)
    timestamp: datetime = field(default_factory=datetime.utcnow)


@dataclass
class SecretAccessLog:
    """密钥访问日志"""
    key: str
    accessed_at: datetime
    accessed_by: str | None = None


@dataclass
class CommandResult:
    """命令执行结果"""
    success: bool = False
    stdout: str = ""
    stderr: str = ""
    exit_code: int = -1


### 7.2 SecurityService 接口

```python
class SecurityService:
    """统一安全服务"""

    def __init__(
        self,
        whitelist_manager: WhitelistManager,
        sandbox_executor: SandboxExecutor,
        dangerous_scanner: DangerousFunctionScanner,
        secret_manager: SecretManager,
    ) -> None:
        self._whitelist = whitelist_manager
        self._sandbox = sandbox_executor
        self._scanner = dangerous_scanner
        self._secrets = secret_manager

    async def execute_with_security(
        self,
        context: SecurityContext,
        tool_name: str,
        **kwargs: Any,
    ) -> Any:
        """在安全检查下执行工具"""
        # 1. 权限检查
        self._check_permissions(context, tool_name)

        # 2. 白名单验证
        if not self._whitelist.is_tool_allowed(tool_name):
            raise SecurityViolationError(f"Tool not allowed: {tool_name}")

        # 3. 危险函数检测（如果是代码执行类工具）
        if self._is_code_tool(tool_name):
            code = kwargs.get("code", "")
            self._scanner.check_and_raise(code)

        # 4. 执行工具
        return await self._execute_tool(context, tool_name, **kwargs)

    def _check_permissions(
        self,
        context: SecurityContext,
        tool_name: str,
    ) -> None:
        """检查工具执行权限"""
        tool_permission_map = {
            "read_file": PermissionLevel.READ,
            "write_file": PermissionLevel.WRITE,
            "edit_file": PermissionLevel.WRITE,
            "bash": PermissionLevel.EXECUTE,
            "execute_code": PermissionLevel.EXECUTE,
        }
        required = tool_permission_map.get(tool_name, PermissionLevel.READ)
        if not context.has_permission(required):
            raise SecurityViolationError(
                f"Insufficient permissions for {tool_name}"
            )

    def _is_code_tool(self, tool_name: str) -> bool:
        """判断是否为代码执行类工具"""
        return tool_name in {"execute_code", "eval_code", "run_python"}

    async def _execute_tool(
        self,
        context: SecurityContext,
        tool_name: str,
        **kwargs: Any,
    ) -> Any:
        """执行工具（带沙箱保护）"""
        # 工具执行逻辑
        pass
```

       │
       ▼
3. WhitelistManager 验证白名单
       │
       ├──► 验证通过 ──► 继续
       │
       └──► 验证失败 ──► 记录日志 + 抛出异常
       │
       ▼
4. SandboxExecutor 执行命令
       │
       ├──► 执行成功 ──► 返回结果
       │
       └──► 执行失败 ──► 记录日志 + 抛出异常
       │
       ▼
5. EventBus.publish("security_event", payload)
       │
       ▼
6. 日志记录（审计追溯）
```

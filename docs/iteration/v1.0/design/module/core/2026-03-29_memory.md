# Memory 模块设计文档

## 1. 模块概述

### 1.1 模块名称

Memory（记忆系统）

### 1.2 职责

Memory 模块是 Mozi AI Coding Agent 上下文层的核心组件，负责：
- 管理短期记忆（滑动窗口内的最近对话内容）
- 管理长期记忆（用户偏好、关键事实、项目知识）
- 提供记忆召回能力，支持根据上下文检索相关记忆
- 提供记忆压缩能力，在上下文超过限制时智能压缩

### 1.3 核心能力

| 能力 | 说明 |
| ---- | ---- |
| 短期记忆管理 | 基于滑动窗口的最近 N 轮对话管理，自动淘汰旧消息 |
| 长期记忆存储 | 基于 embedding 的向量化存储，支持语义检索 |
| 记忆召回 | 根据当前上下文召回相关的短期和长期记忆 |
| 记忆压缩 | 当记忆超过阈值时，自动压缩或归档低价值记忆 |
| 重要性评分 | 对记忆块进行重要性评分，决定保留优先级 |
| 记忆持久化 | 支持记忆的持久化存储，跨会话恢复 |

### 1.4 核心问题

| 问题 | 说明 |
| ---- | ---- |
| 记忆容量限制 | 向量存储有容量限制，需要管理淘汰策略 |
| 检索质量 | 如何召回与当前上下文最相关的记忆 |
| 记忆价值评估 | 如何判断哪些记忆值得保留 |
| 短期 vs 长期 | 如何区分和协同管理短期和长期记忆 |

### 1.5 难点

| 难点 | 说明 |
| ---- | ---- |
| embedding 计算成本 | 高质量 embedding 需要计算资源 |
| 相似度阈值 | 如何设置相似度阈值避免噪声召回 |
| 记忆压缩时机 | 何时触发压缩，压缩比例如何确定 |
| 跨会话记忆传递 | 长期记忆如何在会话间传递和复用 |

### 1.6 解决方案

| 方案 | 说明 |
| ---- | ---- |
| 分层记忆架构 | ShortTermMemory（滑动窗口）+ LongTermMemory（向量存储） |
| 重要性评分 | 基于访问频率、相关性等因素评分 |
| MemoryCompactor | 自动压缩低价值记忆，释放空间 |
| 向量检索 | 使用余弦相似度进行语义检索 |

## 3. 短期记忆

### 3.1 滑动窗口设计

短期记忆采用固定大小的滑动窗口机制：

```
+------------------+  新消息进入
                    |
                    ▼
    ┌─────────────────────────────┐
    │      滑动窗口（窗口大小 N）    │
    │  [msg_N-2] [msg_N-1] [msg_N] │
    └─────────────────────────────┘
                    │
                    ▼ 旧消息移出
```

| 配置项 | 默认值 | 说明 |
| ---- | ---- | ---- |
| `window_size` | 10 | 窗口内保留的消息数 |
| `window_type` | `sliding` | 窗口类型：sliding/rolling |
| `eviction_policy` | `fifo` | 淘汰策略：fifo/lru |

### 3.2 消息管理

```python
class ShortTermMemory:
    """短期记忆管理器"""

    def __init__(
        self,
        session_id: str,
        window_size: int = 10,
        eviction_policy: str = "fifo",
    ) -> None:
        self._session_id = session_id
        self._window_size = window_size
        self._eviction_policy = eviction_policy
        self._messages: list[MemoryBlock] = []

    async def add(self, message: MemoryBlock) -> None:
        """添加消息到短期记忆"""
        self._messages.append(message)
        await self._evict_if_needed()

    async def _evict_if_needed(self) -> None:
        """根据淘汰策略移除旧消息"""
        if len(self._messages) > self._window_size:
            if self._eviction_policy == "fifo":
                self._messages.pop(0)
            elif self._eviction_policy == "lru":
                self._messages.pop(0)  # 简化实现，实际需更新访问时间

    async def get_recent(self, limit: int | None = None) -> list[MemoryBlock]:
        """获取最近的记忆"""
        if limit is None:
            return list(self._messages)
        return self._messages[-limit:]

    async def clear(self) -> None:
        """清空短期记忆"""
        self._messages.clear()
```

### 3.3 短期记忆配置

| ---- | ---- | ---- | ---- |
| `short_term.window_size` | int | 10 | 滑动窗口大小 |
| `short_term.eviction_policy` | string | "fifo" | 淘汰策略 |
| `short_term.max_tokens` | int | 8000 | 最大 token 数 |

## 4. 长期记忆

### 4.1 记忆类型

长期记忆分为三种类型：

| 类型 | 说明 | 存储方式 |
| ---- | ---- | -------- |
| `PREFERENCE` | 用户偏好 | Key-Value 存储 |
| `FACT` | 关键事实 | 向量存储 |
| `KNOWLEDGE` | 项目知识 | 向量存储 |

### 4.2 用户偏好存储

```python
class PreferenceMemory:
    """用户偏好记忆"""

    def __init__(self, storage: KeyValueStore) -> None:
        self._storage = storage

    async def set(self, user_id: str, key: str, value: Any) -> None:
        """设置用户偏好"""
        preference_key = f"pref:{user_id}:{key}"
        await self._storage.set(preference_key, json.dumps(value))

    async def get(self, user_id: str, key: str) -> Any | None:
        """获取用户偏好"""
        preference_key = f"pref:{user_id}:{key}"
        value = await self._storage.get(preference_key)
        return json.loads(value) if value else None

    async def get_all(self, user_id: str) -> dict[str, Any]:
        """获取用户所有偏好"""
        pattern = f"pref:{user_id}:*"
        items = await self._storage.get_pattern(pattern)
        return {k.split(":")[-1]: json.loads(v) for k, v in items}
```

### 4.3 关键事实存储

关键事实以向量形式存储，支持语义检索：

```python
class FactMemory:
    """关键事实记忆"""

    def __init__(
        self,
        vector_store: VectorStore,
        embedding_generator: EmbeddingGenerator,
    ) -> None:
        self._vector_store = vector_store
        self._embedding_generator = embedding_generator

    async def store(
        self,
        session_id: str,
        fact: str,
        metadata: dict[str, Any] | None = None,
    ) -> MemoryBlock:
        """存储关键事实"""
        embedding = await self._embedding_generator.generate(fact)
        block = MemoryBlock(
            id=str(uuid4()),
            session_id=session_id,
            content=fact,
            memory_type=MemoryType.FACT,
            embedding=embedding,
            importance=0.8,  # 事实默认高重要性
            metadata=metadata or {},
        )
        await self._vector_store.upsert(session_id, [block])
        return block

    async def search(
        self,
        query: str,
        session_id: str,
        limit: int = 5,
    ) -> list[MemoryBlock]:
        """搜索相关事实"""
        query_embedding = await self._embedding_generator.generate(query)
        return await self._vector_store.search(
            query_embedding,
            session_id=session_id,
            top_k=limit,
            memory_type=MemoryType.FACT,
        )
```

### 4.4 项目知识存储

项目知识通过文档片段管理：

```python
class KnowledgeMemory:
    """项目知识记忆"""

    def __init__(
        self,
        vector_store: VectorStore,
        embedding_generator: EmbeddingGenerator,
    ) -> None:
        self._vector_store = vector_store
        self._embedding_generator = embedding_generator

    async def index_document(
        self,
        session_id: str,
        document: str,
        chunk_size: int = 500,
        overlap: int = 50,
    ) -> list[MemoryBlock]:
        """将文档切分为块并索引"""
        chunks = self._split_text(document, chunk_size, overlap)
        blocks = []

        for i, chunk in enumerate(chunks):
            embedding = await self._embedding_generator.generate(chunk)
            block = MemoryBlock(
                id=f"{session_id}:chunk:{i}",
                session_id=session_id,
                content=chunk,
                memory_type=MemoryType.KNOWLEDGE,
                embedding=embedding,
                importance=0.6,
                metadata={"chunk_index": i, "total_chunks": len(chunks)},
            )
            blocks.append(block)

        await self._vector_store.upsert(session_id, blocks)
        return blocks

    def _split_text(
        self,
        text: str,
        chunk_size: int,
        overlap: int,
    ) -> list[str]:
        """文本分块"""
        chunks = []
        start = 0
        while start < len(text):
            end = start + chunk_size
            chunks.append(text[start:end])
            start = end - overlap
        return chunks
```

## 5. 记忆召回

### 5.1 召回流程

```
用户查询
    │
    ▼
Retriever.retrieve(query, session_id)
    │
    ├──► Query 分析（提取关键词、意图）
    │
    ├──► 短期记忆召回 ──► get_recent(limit)
    │         │
    │         ▼
    │    [相关短期记忆]
    │
    ├──► 长期记忆召回 ──► VectorStore.search()
    │         │
    │         ▼
    │    [相关长期记忆]
    │
    ├──► 重要性过滤
    │
    ├──► 时间衰减加权
    │
    └──► 排序整合
            │
            ▼
    返回 MemoryBlock 列表
```

### 5.2 检索策略

| 策略 | 适用场景 | 实现方式 |
| ---- | ---- | ---- |
| `semantic` | 语义相关性 | embedding 余弦相似度 |
| `keyword` | 精确关键词匹配 | BM25 / 倒排索引 |
| `hybrid` | 综合场景 | 语义 + 关键词融合 |
| `temporal` | 时间敏感 | 时间衰减权重 |

### 5.3 检索器实现

```python
class MemoryRetriever:
    """记忆检索器"""

    def __init__(
        self,
        short_term: ShortTermMemory,
        long_term: LongTermMemory,
        importance_scorer: ImportanceScorer,
        config: RetrieverConfig,
    ) -> None:
        self._short_term = short_term
        self._long_term = long_term
        self._importance_scorer = importance_scorer
        self._config = config

    async def retrieve(
        self,
        query: str,
        session_id: str,
        memory_types: list[MemoryType] | None = None,
        limit: int = 10,
    ) -> list[MemoryBlock]:
        """
        召回相关记忆

        Args:
            query: 查询文本
            session_id: 会话 ID
            memory_types: 要召回的记忆类型，None 表示全部
            limit: 返回数量限制

        Returns:
            相关记忆列表，按相关性排序
        """
        if memory_types is None:
            memory_types = list(MemoryType)

        results: list[MemoryBlock] = []

        # 召回短期记忆
        if MemoryType.SHORT_TERM in memory_types:
            short_results = await self._retrieve_short_term(query, session_id)
            results.extend(short_results)

        # 召回长期记忆
        if any(t in memory_types for t in [MemoryType.FACT, MemoryType.KNOWLEDGE]):
            long_results = await self._retrieve_long_term(
                query, session_id, memory_types
            )
            results.extend(long_results)

        # 重要性评分和排序
        scored_results = await self._score_and_rank(results, query)

        # 截取限制数量
        return scored_results[:limit]

    async def _retrieve_short_term(
        self,
        query: str,
        session_id: str,
    ) -> list[MemoryBlock]:
        """召回短期记忆"""
        recent = await self._short_term.get_recent(limit=self._config.max_short_term)
        return self._filter_by_relevance(recent, query)

    async def _retrieve_long_term(
        self,
        query: str,
        session_id: str,
        memory_types: list[MemoryType],
    ) -> list[MemoryBlock]:
        """召回长期记忆"""
        results = []
        for memory_type in memory_types:
            if memory_type == MemoryType.SHORT_TERM:
                continue
            type_results = await self._long_term.search(
                query=query,
                session_id=session_id,
                memory_type=memory_type,
                limit=self._config.max_long_term_per_type,
            )
            results.extend(type_results)
        return results

    def _filter_by_relevance(
        self,
        blocks: list[MemoryBlock],
        query: str,
    ) -> list[MemoryBlock]:
        """基于关键词过滤"""
        query_keywords = set(query.lower().split())
        filtered = []
        for block in blocks:
            block_keywords = set(block.content.lower().split())
            if query_keywords & block_keywords:  # 有交集
                filtered.append(block)
        return filtered

    async def _score_and_rank(
        self,
        blocks: list[MemoryBlock],
        query: str,
    ) -> list[MemoryBlock]:
        """评分和排序"""
        scored = []
        for block in blocks:
            importance = await self._importance_scorer.score(block, query)
            recency = self._calculate_recency_score(block)
            combined_score = (
                importance * self._config.importance_weight
                + recency * self._config.recency_weight
            )
            scored.append((combined_score, block))

        scored.sort(key=lambda x: x[0], reverse=True)
        return [block for _, block in scored]

    def _calculate_recency_score(self, block: MemoryBlock) -> float:
        """计算时间衰减分数"""
        now = datetime.now()
        age = (now - block.accessed_at).total_seconds()
        decay = math.exp(-age / self._config.recency_decay_seconds)
        return decay
```

### 5.4 召回配置

| 配置项 | 类型 | 默认值 | 说明 |
| ---- | ---- | ---- | ---- |
| `retriever.max_short_term` | int | 10 | 最大召回短期记忆数 |
| `retriever.max_long_term_per_type` | int | 5 | 每类型最大召回数 |
| `retriever.importance_weight` | float | 0.7 | 重要性权重 |
| `retriever.recency_weight` | float | 0.3 | 时间衰减权重 |

## 6. 记忆压缩

### 6.1 压缩触发条件

| 条件 | 阈值 | 动作 |
| ---- | ---- | ---- |
| 短期记忆超窗口 | `window_size * 1.5` | 触发滑动淘汰 |
| 长期记忆超限 | `max_memories` | 触发压缩或归档 |
| 上下文超 Token | `max_context_tokens` | 触发整体压缩 |
| 记忆过期 | `ttl_days` | 触发归档或删除 |

### 6.2 压缩策略

```python
class MemoryCompactor:
    """记忆压缩器"""

    def __init__(
        self,
        importance_scorer: ImportanceScorer,
        summarizer: Summarizer,
        config: CompactorConfig,
    ) -> None:
        self._importance_scorer = importance_scorer
        self._summarizer = summarizer
        self._config = config

    async def compact(
        self,
        memories: list[MemoryBlock],
        target_count: int | None = None,
        target_tokens: int | None = None,
    ) -> list[MemoryBlock]:
        """
        压缩记忆列表

        Args:
            memories: 原始记忆列表
            target_count: 目标记忆数量
            target_tokens: 目标 token 数

        Returns:
            压缩后的记忆列表
        """
        if not memories:
            return memories

        # 计算保留优先级
        scored = []
        for mem in memories:
            score = await self._importance_scorer.score(mem, query="")
            scored.append((score, mem))

        scored.sort(key=lambda x: x[0], reverse=True)

        # 选择保留的记忆
        if target_count:
            kept = [mem for _, mem in scored[:target_count]]
        else:
            kept = list(memories)

        # Token 限制检查
        if target_tokens:
            kept = self._trim_to_token_limit(kept, target_tokens)

        return kept

    async def summarize(
        self,
        memories: list[MemoryBlock],
    ) -> MemoryBlock:
        """将多个记忆合并为一个摘要"""
        contents = [m.content for m in memories]
        summary_text = await self._summarizer.summarize(contents)

        return MemoryBlock(
            id=str(uuid4()),
            session_id=memories[0].session_id if memories else "",
            content=summary_text,
            memory_type=MemoryType.SUMMARY,
            importance=0.9,  # 摘要保持高重要性
            metadata={
                "source_count": len(memories),
                "source_ids": [m.id for m in memories],
            },
        )

    def _trim_to_token_limit(
        self,
        memories: list[MemoryBlock],
        max_tokens: int,
    ) -> list[MemoryBlock]:
        """按 token 限制裁剪"""
        result = []
        total_tokens = 0

        for mem in memories:
            tokens = self._estimate_tokens(mem)
            if total_tokens + tokens <= max_tokens:
                result.append(mem)
                total_tokens += tokens
            else:
                break

        return result

    def _estimate_tokens(self, block: MemoryBlock) -> int:
        """估算 token 数（简化版）"""
        return len(block.content) // 4
```

### 6.3 压缩策略类型

| 策略 | 说明 | 适用场景 |
| ---- | ---- | -------- |
| `importance` | 基于重要性评分保留 | 通用场景 |
| `summarize` | 将低重要性记忆合并摘要 | 信息密度低 |
| `prune` | 直接删除低于阈值记忆 | 快速释放空间 |
| `archive` | 将低价值记忆归档到冷存储 | 长期积累 |

## 7. 接口设计

### 7.1 核心数据模型

```python
from dataclasses import dataclass, field
from datetime import datetime
from enum import Enum
from typing import Any
from uuid import uuid4


class MemoryType(Enum):
    """记忆类型"""
    SHORT_TERM = "short_term"
    LONG_TERM = "long_term"
    PREFERENCE = "preference"
    FACT = "fact"
    KNOWLEDGE = "knowledge"
    SUMMARY = "summary"


class MemoryStatus(Enum):
    """记忆状态"""
    ACTIVE = "active"
    ARCHIVED = "archived"
    DELETED = "deleted"


@dataclass
class MemoryBlock:
    """记忆块"""
    id: str
    session_id: str
    content: str
    memory_type: MemoryType
    embedding: list[float] | None = None
    importance: float = 0.5  # 0.0 - 1.0
    status: MemoryStatus = MemoryStatus.ACTIVE
    created_at: datetime = field(default_factory=datetime.now)
    accessed_at: datetime = field(default_factory=datetime.now)
    expires_at: datetime | None = None
    metadata: dict[str, Any] = field(default_factory=dict)

    def __post_init__(self) -> None:
        if self.id == "":
            self.id = str(uuid4())


@dataclass
class MemoryConfig:
    """记忆配置"""
    short_term_window_size: int = 10
    short_term_max_tokens: int = 8000
    long_term_max_memories: int = 1000
    long_term_ttl_days: int = 90
    compaction_threshold: float = 0.8
    default_importance: float = 0.5


@dataclass
class RetrievalResult:
    """检索结果"""
    blocks: list[MemoryBlock]
    total_count: int
    query: str
    session_id: str
    retrieval_time_ms: float
    metadata: dict[str, Any] = field(default_factory=dict)
```

### 7.2 Memory 抽象基类

```python
from abc import ABC, abstractmethod


class Memory(ABC):
    """Memory 模块抽象基类"""

    @abstractmethod
    async def remember(
        self,
        session_id: str,
        content: str,
        memory_type: MemoryType,
        importance: float | None = None,
        metadata: dict[str, Any] | None = None,
    ) -> MemoryBlock:
        """
        存储记忆

        Args:
            session_id: 会话 ID
            content: 记忆内容
            memory_type: 记忆类型
            importance: 重要性评分（None 使用默认值）
            metadata: 额外元数据

        Returns:
            创建的记忆块
        """

    @abstractmethod
    async def recall(
        self,
        query: str,
        session_id: str,
        memory_types: list[MemoryType] | None = None,
        limit: int = 10,
    ) -> list[MemoryBlock]:
        """
        召回相关记忆

        Args:
            query: 查询文本
            session_id: 会话 ID
            memory_types: 记忆类型过滤（None 表示全部）
            limit: 返回数量限制

        Returns:
            相关记忆列表
        """

    @abstractmethod
    async def get(
        self,
        memory_id: str,
        session_id: str,
    ) -> MemoryBlock | None:
        """
        获取指定记忆

        Args:
            memory_id: 记忆 ID
            session_id: 会话 ID

        Returns:
            记忆块，不存在返回 None
        """

    @abstractmethod
    async def forget(
        self,
        memory_id: str,
        session_id: str,
    ) -> bool:
        """
        删除记忆

        Args:
            memory_id: 记忆 ID
            session_id: 会话 ID

        Returns:
            是否删除成功
        """

    @abstractmethod
    async def compact(self, session_id: str) -> int:
        """
        压缩会话记忆

        Args:
            session_id: 会话 ID

        Returns:
            压缩后删除的记忆数量
        """
```

### 7.3 Memory 实现类

```python
class MemoryManager(Memory):
    """记忆管理器实现"""

    def __init__(
        self,
        short_term: ShortTermMemory,
        long_term: LongTermMemory,
        retriever: MemoryRetriever,
        compactor: MemoryCompactor,
        storage: MemoryStorage,
        config: MemoryConfig,
    ) -> None:
        self._short_term = short_term
        self._long_term = long_term
        self._retriever = retriever
        self._compactor = compactor
        self._storage = storage
        self._config = config

    async def remember(
        self,
        session_id: str,
        content: str,
        memory_type: MemoryType,
        importance: float | None = None,
        metadata: dict[str, Any] | None = None,
    ) -> MemoryBlock:
        """存储记忆"""
        # 创建记忆块
        block = MemoryBlock(
            id=str(uuid4()),
            session_id=session_id,
            content=content,
            memory_type=memory_type,
            importance=importance or self._config.default_importance,
            metadata=metadata or {},
        )

        # 路由到对应存储
        if memory_type == MemoryType.SHORT_TERM:
            await self._short_term.add(block)
        else:
            await self._long_term.store(block)

        # 持久化
        await self._storage.save(block)

        return block

    async def recall(
        self,
        query: str,
        session_id: str,
        memory_types: list[MemoryType] | None = None,
        limit: int = 10,
    ) -> list[MemoryBlock]:
        """召回相关记忆"""
        return await self._retriever.retrieve(
            query=query,
            session_id=session_id,
            memory_types=memory_types,
            limit=limit,
        )

    async def get(
        self,
        memory_id: str,
        session_id: str,
    ) -> MemoryBlock | None:
        """获取指定记忆"""
        return await self._storage.load(memory_id, session_id)

    async def forget(
        self,
        memory_id: str,
        session_id: str,
    ) -> bool:
        """删除记忆"""
        block = await self._storage.load(memory_id, session_id)
        if not block:
            return False

        block.status = MemoryStatus.DELETED
        await self._storage.save(block)

        # 从对应存储移除
        if block.memory_type == MemoryType.SHORT_TERM:
            await self._short_term.remove(memory_id)
        else:
            await self._long_term.remove(memory_id)

        return True

    async def compact(self, session_id: str) -> int:
        """压缩会话记忆"""
        memories = await self._storage.load_session(session_id)
        original_count = len(memories)

        compacted = await self._compactor.compact(
            memories,
            target_count=self._config.long_term_max_memories,
        )

        # 保存压缩后的记忆
        for mem in compacted:
            await self._storage.save(mem)

        return original_count - len(compacted)
```

### 7.4 存储接口

```python
class MemoryStorage(ABC):
    """记忆存储抽象接口"""

    @abstractmethod
    async def save(self, block: MemoryBlock) -> None:
        """保存记忆块"""
        ...

    @abstractmethod
    async def load(self, memory_id: str, session_id: str) -> MemoryBlock | None:
        """加载记忆块"""
        ...

    @abstractmethod
    async def load_session(
        self,
        session_id: str,
        memory_type: MemoryType | None = None,
    ) -> list[MemoryBlock]:
        """加载会话所有记忆"""
        ...

    @abstractmethod
    async def delete(self, memory_id: str, session_id: str) -> bool:
        """删除记忆"""
        ...

    @abstractmethod
    async def exists(self, memory_id: str, session_id: str) -> bool:
        """检查记忆是否存在"""
        ...


class FileMemoryStorage(MemoryStorage):
    """文件存储实现"""

    def __init__(self, storage_dir: str = "~/.mozi/memory") -> None:
        self._storage_dir = Path(storage_dir).expanduser()

    async def save(self, block: MemoryBlock) -> None:
        """保存到文件"""
        session_dir = self._storage_dir / block.session_id
        session_dir.mkdir(parents=True, exist_ok=True)
        file_path = session_dir / f"{block.id}.json"
        await write_json(file_path, self._serialize(block))

    async def load(self, memory_id: str, session_id: str) -> MemoryBlock | None:
        """从文件加载"""
        file_path = self._storage_dir / session_id / f"{memory_id}.json"
        if not file_path.exists():
            return None
        data = await read_json(file_path)
        return self._deserialize(data)

    async def load_session(
        self,
        session_id: str,
        memory_type: MemoryType | None = None,
    ) -> list[MemoryBlock]:
        """加载会话所有记忆"""
        session_dir = self._storage_dir / session_id
        if not session_dir.exists():
            return []

        blocks = []
        for file_path in session_dir.glob("*.json"):
            data = await read_json(file_path)
            block = self._deserialize(data)
            if memory_type is None or block.memory_type == memory_type:
                blocks.append(block)

        return blocks

    async def delete(self, memory_id: str, session_id: str) -> bool:
        """删除文件"""
        file_path = self._storage_dir / session_id / f"{memory_id}.json"
        if file_path.exists():
            file_path.unlink()
            return True
        return False

    async def exists(self, memory_id: str, session_id: str) -> bool:
        """检查文件是否存在"""
        return (self._storage_dir / session_id / f"{memory_id}.json").exists()
```

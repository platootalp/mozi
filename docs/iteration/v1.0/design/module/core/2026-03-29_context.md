# Context 模块设计文档

## 1. 模块概述

### 1.1 模块名称

Context（上下文层）

### 1.2 职责

Context 模块是 Mozi AI Coding Agent 的上下文管理层，负责：
- 构建当前任务的上下文信息
- 管理历史消息与会话状态
- 提供向量化检索能力，从 Memory 召回相关记忆
- 整合短期记忆（滑动窗口）和长期记忆（向量化存储）

### 1.3 核心能力

| 能力 | 说明 |
| --- | --- |
| 上下文构建 | 聚合用户输入、历史消息、Memory 数据，生成完整的模型调用上下文 |
| 历史管理 | 管理多轮对话历史，支持上下文窗口滑动 |
| 向量化检索 | 基于 embedding 从长期记忆召回相关内容 |
| 上下文压缩 | 当上下文超过阈值时，自动压缩或截断 |
| 会话隔离 | 支持多会话并发，上下文相互隔离 |

### 1.4 核心问题

| 问题 | 说明 |
| ---- | ---- |
| 上下文长度限制 | 模型有固定上下文窗口，多轮对话后可能超出限制 |
| 信息检索效率 | 从大量历史记忆中快速找到相关内容 |
| 上下文构建复杂性 | 需要整合用户输入、历史消息、记忆数据 |
| 上下文压缩质量 | 压缩后需保留关键信息，不丢失重要上下文 |

### 1.5 难点

| 难点 | 说明 |
| ---- | ---- |
| 向量化检索精度 | embedding 模型的选择和相似度计算影响检索质量 |
| 滑动窗口策略 | 如何决定保留哪些消息、淘汰哪些消息 |
| 上下文压缩算法 | 摘要/截断算法需要平衡信息密度和完整性 |
| 多会话上下文隔离 | 不同会话的上下文需要独立管理 |

### 1.6 解决方案

| 方案 | 说明 |
| ---- | ---- |
| 向量检索 | 使用 embedding 模型将记忆向量化，支持语义相似度检索 |
| 滑动窗口 | 基于 token 计数器的滑动窗口，自动管理上下文容量 |
| 混合压缩策略 | 保留关键事实和最近对话，摘要中间历史 |
| 会话级别隔离 | 每个会话独立上下文对象，通过 session_id 隔离 |

## 2. 模块结构

### 2.1 目录结构

```

    __init__.py

    compactor.py            # 上下文压缩器
```

### 2.2 关键文件说明

| 文件 | 职责 |
| --- | --- |
| `context/builder.py` | 上下文构建入口，协调 Session/Memory 数据整合 |

| `context/retriever.py` | 检索器实现，基于 embedding 相似度召回 |
| `context/models.py` | 定义 ContextBlock、BuiltContext 等数据模型 |

## 3. 上下文构建

### 3.1 构建流程

```
用户输入 + session_id
    │
    ▼
ContextBuilder.build()
    │
    ├──► 从 SessionManager 获取会话历史
    │
    ├──► 从 Memory 召回短期记忆（滑动窗口）
    │
    ├──► 从 Memory 召回长期记忆（向量化检索）
    │
    ├──► 整合用户输入 + 历史 + 记忆
    │
    ├──► 上下文压缩（必要时）
    │
    ▼
返回完整上下文（ContextBlock）
    │
    ▼
Model.invoke(context)
```

### 3.2 上下文组成

上下文由以下部分组成：

| 组成部分 | 来源 | 说明 |
| --- | --- | --- |
| System Prompt | 配置 | 固定的系统级提示词 |
| 用户输入 | 当前请求 | 用户最新输入 |
| 对话历史 | SessionManager | 最近 N 轮对话（可配置窗口） |
| 短期记忆 | Memory (短期) | 滑动窗口内的重要信息 |
| 长期记忆 | Memory (长期) | 向量化检索召回的相关记忆 |
| 工具结果 | 执行记录 | 最近工具调用结果（可选） |

### 3.3 上下文窗口管理

上下文窗口大小通过以下机制控制：

| 维度 | 默认值 | 配置项 | 说明 |
| --- | --- | --- | --- |
| 对话历史轮数 | 10 | context.window_messages | 保留的对话轮数 |
| 单条消息最大长度 | 2000 | context.max_message_length | 超出则截断 |
| 上下文总 token 上限 | 16000 | context.max_tokens | 超出则压缩 |
| 召回记忆数量 | 5 | context.max_memories | 向量检索召回数 |

## 4. 向量化检索

### 4.1 检索流程

```
用户输入 embedding
    │
    ▼
Retriever.retrieve(query_embedding, session_id)
    │
    ├──► 从 VectorStore 查询相似向量
    │
    ├──► 过滤不相关的记忆（相似度 < threshold）
    │
    ├──► 按相似度排序
    │
    ├──► 截取 top-K 结果
    │
    ▼
返回相关记忆列表（list[MemoryBlock]）
```

### 4.2 向量存储抽象

```python
class VectorStore(ABC):
    """向量存储抽象接口"""

    @abstractmethod
    async def upsert(
        self,
        session_id: str,
        blocks: list[MemoryBlock],
    ) -> None:
        """
        插入或更新记忆块

        Args:
            session_id: 会话 ID
            blocks: 记忆块列表
        """

    @abstractmethod
    async def search(
        self,
        query_embedding: list[float],
        session_id: str,
        top_k: int = 5,
        threshold: float = 0.7,
    ) -> list[MemoryBlock]:
        """
        搜索相似记忆

        Args:
            query_embedding: 查询向量
            session_id: 会话 ID
            top_k: 返回数量
            threshold: 相似度阈值

        Returns:
            匹配的记忆块列表
        """

    @abstractmethod
    async def delete_session(self, session_id: str) -> None:
        """删除会话的所有记忆"""
```

### 4.3 检索策略

| 策略 | 适用场景 | 实现方式 |
| --- | --- | --- |
| 相似度检索 | 通用场景 | embedding 余弦相似度 |
| 关键词过滤 | 明确实体 | 先 BM25 再向量检索 |
| 时间衰减 | 历史偏好 | 结合时间权重的相似度 |
| 会话隔离 | 多会话 | 按 session_id 隔离检索 |

```python
from dataclasses import dataclass, field
from datetime import datetime
from enum import Enum
from typing import Any


class MessageRole(Enum):
    """消息角色"""
    USER = "user"
    ASSISTANT = "assistant"
    SYSTEM = "system"
    TOOL = "tool"


class MemoryType(Enum):
    """记忆类型"""
    SHORT_TERM = "short_term"
    LONG_TERM = "long_term"
    EPISODIC = "episodic"
    SEMANTIC = "semantic"


@dataclass
class Message:
    """对话消息"""
    id: str
    role: MessageRole
    content: str
    created_at: datetime = field(default_factory=datetime.now)
    metadata: dict[str, Any] = field(default_factory=dict)


@dataclass
class MemoryBlock:
    """记忆块"""
    id: str
    session_id: str
    content: str
    memory_type: MemoryType
    embedding: list[float] | None = None
    importance: float = 0.5  # 0.0 - 1.0
    created_at: datetime = field(default_factory=datetime.now)
    accessed_at: datetime = field(default_factory=datetime.now)
    metadata: dict[str, Any] = field(default_factory=dict)


@dataclass
class ContextBlock:
    """上下文块"""
    role: MessageRole
    content: str
    metadata: dict[str, Any] = field(default_factory=dict)


@dataclass
class BuiltContext:
    """构建完成的上下文"""
    blocks: list[ContextBlock]
    total_tokens: int
    memory_sources: list[str] = field(default_factory=list)
    truncated: bool = False
    metadata: dict[str, Any] = field(default_factory=dict)
```

### 5.5 上下文窗口实现

```python
@dataclass
class ContextWindowConfig:
    """上下文窗口配置"""
    max_tokens: int = 100000          # 最大 Token 数
    max_messages: int = 1000           # 最大消息数
    window_type: str = "sliding"       # 窗口类型: sliding/semantic
    truncation_strategy: str = "preserve_last"  # 截断策略


class ContextWindow:
    """上下文窗口管理器"""

    def __init__(
        self,
        config: ContextWindowConfig,
        compactor: "ContextCompactor",
    ) -> None:
        self._config = config
        self._compactor = compactor

    def is_within_limit(self, session: "Session") -> bool:
        """检查是否在限制内"""
        total_tokens = self._estimate_tokens(session)
        return (
            total_tokens <= self._config.max_tokens
            and len(session.messages) <= self._config.max_messages
        )

    def truncate(
        self,
        session: "Session",
    ) -> "Session":
        """截断会话到限制内"""
        if self.is_within_limit(session):
            return session
        return self._preserve_last_truncation(session)

    def _preserve_last_truncation(
        self,
        session: "Session",
    ) -> "Session":
        """保留最后 N 条消息的截断策略"""
        max_tokens = self._config.max_tokens
        max_messages = self._config.max_messages
        preserved_messages: list[Message] = []
        accumulated_tokens = 0

        for msg in reversed(session.messages):
            msg_tokens = self._estimate_message_tokens(msg)
            if (
                accumulated_tokens + msg_tokens > max_tokens
                or len(preserved_messages) >= max_messages
            ):
                break
            preserved_messages.insert(0, msg)
            accumulated_tokens += msg_tokens

        return Session(
            id=session.id,
            metadata=session.metadata,
            messages=preserved_messages,
            status=session.status,
            parent_session_id=session.parent_session_id,
        )

    def _estimate_tokens(self, session: "Session") -> int:
        """估算会话 token 数"""
        estimator = TokenEstimator()
        return estimator.estimate_messages(session.messages)

    def _estimate_message_tokens(self, msg: Message) -> int:
        """估算单条消息 token 数"""
        estimator = TokenEstimator()
        return estimator.estimate(msg.content) + len(msg.role.value) + 1


class TokenEstimator:
    """Token 估算器（简化版，实际使用 tiktoken）"""

    CHARS_PER_TOKEN = 4

    def estimate(self, text: str) -> int:
        """估算文本 Token 数"""
        return len(text) // self.CHARS_PER_TOKEN

    def estimate_messages(self, messages: list[Message]) -> int:
        """估算消息列表总 Token 数"""
        total = 0
        for msg in messages:
            role_prefix = len(msg.role.value) + 1
            total += self.estimate(msg.content) + role_prefix
        return total
```

### 5.6 上下文压缩器实现

```python
class ContextCompactor:
    """上下文压缩器"""

    def __init__(
        self,
        model_adapter: "ModelAdapter",
        preservation_strategy: str = "hybrid",
    ) -> None:
        self._model_adapter = model_adapter
        self._strategy = preservation_strategy

    async def compact(
        self,
        messages: list[Message],
        target_tokens: int,
    ) -> list[Message]:
        """
        压缩消息列表到目标 Token 数

        Args:
            messages: 原始消息列表
            target_tokens: 目标 Token 数

        Returns:
            压缩后的消息列表
        """
        if self._strategy == "hybrid":
            return await self._hybrid_compact(messages, target_tokens)
        elif self._strategy == "semantic":
            return await self._semantic_compact(messages, target_tokens)
        else:
            return await self._simple_compact(messages, target_tokens)

    async def _hybrid_compact(
        self,
        messages: list[Message],
        target_tokens: int,
    ) -> list[Message]:
        """混合压缩：保留首尾消息，中间消息摘要"""
        if len(messages) <= 4:
            return messages

        system_msg = messages[0] if messages[0].role == MessageRole.SYSTEM else None
        preserved_tail = messages[-2:] if len(messages) >= 2 else messages[-1:]
        middle_messages = messages[1:-2] if len(messages) > 2 else []

        if not middle_messages:
            result = []
            if system_msg:
                result.append(system_msg)
            result.extend(preserved_tail)
            return result

        summary = await self._summarize(middle_messages)
        result = []
        if system_msg:
            result.append(system_msg)
        result.append(summary)
        result.extend(preserved_tail)
        return result

    async def _summarize(self, messages: list[Message]) -> Message:
        """使用 LLM 生成摘要"""
        content = self._build_summary_prompt(messages)
        response = await self._model_adapter.invoke(
            messages=[Message(role=MessageRole.USER, content=content)],
        )
        return Message(
            role=MessageRole.SYSTEM,
            content=f"[Summary of earlier conversation]\n{response.content}",
        )

    def _build_summary_prompt(self, messages: list[Message]) -> str:
        """构建摘要提示"""
        conversation_text = "\n".join(
            f"{msg.role.value}: {msg.content[:200]}..."
            if len(msg.content) > 200
            else f"{msg.role.value}: {msg.content}"
            for msg in messages
        )
        return (
            "Summarize the following conversation concisely, "
            "preserving key facts, decisions, and important context:\n\n"
            f"{conversation_text}\n\n"
            "Provide a brief summary (2-3 sentences):"
        )

    async def _semantic_compact(
        self,
        messages: list[Message],
        target_tokens: int,
    ) -> list[Message]:
        """语义压缩：基于重要性保留消息"""
        raise NotImplementedError("Semantic compaction not yet implemented")

    async def _simple_compact(
        self,
        messages: list[Message],
        target_tokens: int,
    ) -> list[Message]:
        """简单压缩：直接截断"""
        estimator = TokenEstimator()
        result = []
        accumulated = 0

        for msg in messages:
            tokens = estimator.estimate(msg.content) + len(msg.role.value) + 1
            if accumulated + tokens > target_tokens:
                break
            result.append(msg)
            accumulated += tokens

        return result
```

## 6. 数据流

### 6.1 与 Orchestrator 的交互

```
Orchestrator.process()
    │
    ▼
ContextBuilder.build(user_input, session_id)
    │
    ▼
BuiltContext 返回
    │
    ▼
Model.invoke(context)
```

Orchestrator 调用 Context.build() 获取完整上下文后，传递给 Model 进行推理。

### 6.2 与 Memory 的交互

```
ContextBuilder.build()
    │
    ▼
Retriever.retrieve(query, session_id)
    │
    ├──► VectorStore.search()
    │
    ▼
List[MemoryBlock]
    │
    ▼
整合到 BuiltContext
```

Context 通过 Retriever 接口访问 Memory 的数据，不直接依赖具体实现。

### 6.3 与 Session 的交互

```
Orchestrator
    │
    ├──► session_manager.create() ──► 创建新会话
    │
    ├──► context_builder.build() ──► 构建完整上下文
    │         │
    │         ▼
    │    SessionManager.get() ──► 获取会话数据
    │         │
    │         ▼
    │    Memory.recall() ──► 召回长期记忆（可选）
    │         │
    │         ▼
    │    ContextBuilder._inject_memories() ──► 注入记忆
    │         │
    │         ▼
    │    ContextWindow.is_within_limit() ──► 检查限制
    │         │
    │         ▼（超出限制）
    │    ContextCompactor.compact() ──► 压缩上下文
    │
    ├──► session_manager.update() ──► 保存更新
    │
    └──► Session.append() ──► 追加消息
              │
              ▼
         延迟保存缓冲
              │
              ▼
         Storage.save() ──► 持久化到磁盘
```

Session 模块提供会话历史管理能力，Context 模块通过 SessionManager 获取历史消息，
通过 ContextBuilder 整合后返回完整上下文。

详细设计见 [Session 模块设计文档](./2026-03-29_session.md)。

### 6.4 与 Model 的交互

```
ContextBuilder.build()
    │
    ▼
BuiltContext
    │
    ▼
Model.invoke(context)
    │
    ▼
Model 返回响应
```

Context 构建的 BuiltContext 作为 Model.invoke() 的输入参数。

### 6.5 完整数据流

```
用户输入
    │
    ▼
CLI / Ingress
    │
    ▼
EventBus.publish("user_message", payload)
    │
    ▼
Orchestrator（意图识别 + 复杂度评分）
    │
    ▼
ContextBuilder.build()

    ├──► SessionContextManager.get_history()
    ├──► Retriever.retrieve() ──► Memory

    ▼
BuiltContext（完整上下文）
    │
    ▼

    │
    ▼
响应生成 + 工具调用
    │
    ▼
SessionContextManager.append() ──► 更新会话历史
    │
    ▼

```

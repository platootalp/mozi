class TestConfigLoader:

| 测试类型 | 执行频率 | 覆盖场景 |
| -------- | -------- | -------- |
| 端到端测试 | 每次 PR | 完整用户交互流程 |
| 多Agent协作测试 | 每次 PR | COMPLEX 任务的多Agent模式 |
| 并发测试 | 每日 | 多会话并发、竞态条件 |
| 性能测试 | 每周 | 响应时间、吞吐量、内存使用 |

### 15.2 安全专项测试

| 测试项 | 工具 | 目标 |
| ------ | ---- | ---- |
| 硬编码密钥扫描 | truffleHog | 每次 PR |
| 漏洞扫描 | pip-audit | 每次构建 |
| 依赖审计 | bandit | 每次构建 |

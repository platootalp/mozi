# Model 模块设计文档

## 1. 模块概述

### 1.1 模块名称

Model（模型适配模块）

### 1.2 职责

Model 模块是 Mozi AI Coding Agent 的核心能力层之一，负责：
- 提供统一的大语言模型调用接口
- 支持多模型适配（Anthropic/OpenAI）
- 管理 Prompt 模板与变量替换
- 处理模型响应的结构化解析

> **注意**：模型选择由 Orchestrator 在路由阶段根据复杂度选择 Agent 时一并决定，Agent 内部绑定特定模型。Model 模块不负责模型选择，只负责执行模型调用。

### 1.3 核心能力

| 能力 | 说明 |
| ---- | ---- |
| 多模型适配 | 支持 Anthropic Claude 系列、OpenAI GPT 系列 |
| 统一接口 | ModelAdapter 抽象接口，统一调用方式 |
| Prompt 模板 | 模板管理、变量替换、格式化 |
| 响应解析 | 结构化解析模型输出（文本/工具调用） |
| 重试机制 | 指数退避重试、熔断保护 |
| 密钥管理 | 安全访问模型 API 密钥 |

### 1.4 核心问题

| 问题 | 说明 |
| ---- | ---- |
| 多模型支持 | 不同模型有不同 API 和响应格式 |
| 模型切换 | 如何在不修改业务代码的情况下切换模型 |
| API 稳定性 | 模型 API 变化导致调用失败 |
| 密钥安全 | 如何安全地管理和传递 API 密钥 |

### 1.5 难点

| 难点 | 说明 |
| ---- | ---- |
| 统一接口抽象 | 不同模型的接口差异如何统一抽象 |
| 响应解析 | 不同模型的响应格式不同，需要适配 |
| 熔断降级 | 模型 API 不可用时如何快速失败 |
| Prompt 兼容性 | 不同模型对 Prompt 格式要求不同 |

### 1.6 解决方案

| 方案 | 说明 |
| ---- | ---- |
| ModelAdapter 抽象 | 统一接口定义，AnthropicAdapter、OpenAIAdapter 实现 |
| 模板管理 | PromptTemplateManager 统一管理模板 |
| Resilience 集成 | 集成 CircuitBreaker、RetryPolicy 提高稳定性 |
| 环境变量注入 | 密钥通过环境变量注入，禁止硬编码 |

## 4. Prompt 模板

### 4.1 模板管理架构

```
PromptTemplateManager
    ├── 模板存储 (文件系统 / 数据库)
    ├── 模板加载 (延迟加载、缓存)
    ├── 变量替换 (安全替换)
    └── 模板验证 (Schema 校验)
```

### 4.2 模板格式

使用 Jinja2 风格的模板语法：

```jinja2
## 系统模板
你是一个专业的 AI 编程助手，擅长{{ language }}开发。

## 用户模板
请帮我{{ action }}以下代码：
{{ code }}

## 工具调用模板
{% if tool_calls %}
我将使用以下工具来完成任务：
{% for tool in tool_calls %}
- {{ tool.name }}: {{ tool.description }}
{% endfor %}
{% endif %}

## 响应格式
{% if format == "json" %}
请以 JSON 格式返回结果。
{% elif format == "markdown" %}
请用 Markdown 格式返回结果。
{% endif %}
```

### 4.4 模板管理接口

```python
class PromptTemplateManager:
    """Prompt 模板管理器"""

    def __init__(self, template_dir: str | None = None) -> None:
        self._template_dir = template_dir or DEFAULT_TEMPLATE_DIR
        self._templates: dict[str, PromptTemplate] = {}
        self._cache: dict[str, str] = {}

    def load_template(self, name: str) -> PromptTemplate:
        """加载指定名称的模板"""
        if name in self._templates:
            return self._templates[name]

        template_path = self._resolve_path(name)
        template_content = self._read_template(template_path)

        template = PromptTemplate(
            name=name,
            content=template_content,
            variables=self._extract_variables(template_content),
        )
        self._templates[name] = template
        return template

    def render(
        self,
        template_name: str,
        variables: dict[str, Any],
    ) -> str:
        """渲染模板，替换变量"""
        template = self._get_cached_template(template_name)
        return template.render(**variables)

    def _extract_variables(self, content: str) -> list[str]:
        """从模板内容提取变量名列表"""
        import re
        pattern = r"\{\{\s*(\w+)(?:\.\w+)*\s*\}\}"
        return list(set(re.findall(pattern, content)))


@dataclass
class PromptTemplate:
    """Prompt 模板"""
    name: str
    content: str
    variables: list[str]

    def render(self, **kwargs: Any) -> str:
        """渲染模板"""
        import re

        def replacer(match: re.Match) -> str:
            key = match.group(1)
            if key in kwargs:
                value = kwargs[key]
                if isinstance(value, list):
                    return ", ".join(str(v) for v in value)
                return str(value)
            return match.group(0)

        return re.sub(r"\{\{\s*\w+(?:\.\w+)*\s*\}\}", replacer, self.content)
```

## 5. 接口设计

### 5.1 核心类型定义

```python
from __future__ import annotations

from abc import ABC, abstractmethod
from dataclasses import dataclass, field
from enum import Enum
from typing import Any


class ModelProvider(Enum):
    """模型提供商枚举"""
    ANTHROPIC = "anthropic"
    OPENAI = "openai"


class MessageRole(Enum):
    """消息角色枚举"""
    SYSTEM = "system"
    USER = "user"
    ASSISTANT = "assistant"
    TOOL = "tool"


@dataclass
class Message:
    """对话消息"""
    role: MessageRole
    content: str
    name: str | None = None
    tool_calls: list[ToolCall] | None = None
    tool_call_id: str | None = None


@dataclass
class ToolCall:
    """工具调用"""
    id: str
    name: str
    arguments: dict[str, Any]


@dataclass
class ModelRequest:
    """模型调用请求"""
    model: str
    messages: list[Message]
    system_prompt: str | None = None
    temperature: float = 1.0
    max_tokens: int | None = None
    top_p: float | None = None
    stop_sequences: list[str] | None = None
    tools: list[dict[str, Any]] | None = None
    stream: bool = False
    metadata: dict[str, Any] = field(default_factory=dict)


@dataclass
class ModelResponse:
    """模型调用响应"""
    content: str
    model: str
    stop_reason: str | None = None
    tool_calls: list[ToolCall] | None = None
    usage: ModelUsage | None = None
    metadata: dict[str, Any] = field(default_factory=dict)


@dataclass
class ModelUsage:
    """模型使用量"""
    input_tokens: int = 0
    output_tokens: int = 0
    total_tokens: int = 0


@dataclass
class ModelInfo:
    """模型信息"""
    name: str
    provider: ModelProvider
    display_name: str
    context_window: int
    tier: str  # "fast" | "balanced" | "powerful"
    supports_tools: bool = True
    supports_vision: bool = False
```

### 5.2 ModelAdapter 抽象基类

```python
class ModelAdapter(ABC):
    """模型适配器抽象基类"""

    @property
    @abstractmethod
    def provider(self) -> ModelProvider:
        """返回模型提供商"""
        raise NotImplementedError

    @property
    @abstractmethod
    def supported_models(self) -> list[ModelInfo]:
        """返回支持的模型列表"""
        raise NotImplementedError

    @abstractmethod
    async def invoke(self, request: ModelRequest) -> ModelResponse:
        """调用模型"""
        raise NotImplementedError

    @abstractmethod
    async def invoke_stream(
        self,
        request: ModelRequest,
    ) -> AsyncIterator[ModelStreamEvent]:
        """流式调用模型"""
        raise NotImplementedError

    @abstractmethod
    def validate_request(self, request: ModelRequest) -> None:
        """验证请求参数"""
        raise NotImplementedError

    @abstractmethod
    def parse_response(self, raw_response: Any) -> ModelResponse:
        """解析原始响应"""
        raise NotImplementedError


@dataclass
class ModelStreamEvent:
    """流式响应事件"""
    type: str  # "content_block", "message_delta", "message_start", etc.
    data: dict[str, Any]
```

### 5.3 AnthropicAdapter 实现

```python
class AnthropicAdapter(ModelAdapter):
    """Anthropic 模型适配器"""

    def __init__(
        self,
        api_key: str | None = None,
        base_url: str | None = None,
        timeout: int = 60,
        max_retries: int = 3,
    ) -> None:
        self._api_key = api_key or self._load_api_key()
        self._base_url = base_url or "https://api.anthropic.com"
        self._timeout = timeout
        self._max_retries = max_retries
        self._client = httpx.AsyncClient(timeout=timeout)

    @property
    def provider(self) -> ModelProvider:
        return ModelProvider.ANTHROPIC

    @property
    def supported_models(self) -> list[ModelInfo]:
        return [
            ModelInfo(
                name="claude-3-5-sonnet-latest",
                provider=ModelProvider.ANTHROPIC,
                display_name="Claude 3.5 Sonnet",
                context_window=200_000,
                tier="balanced",
                supports_tools=True,
            ),
            ModelInfo(
                name="claude-3-5-haiku-latest",
                provider=ModelProvider.ANTHROPIC,
                display_name="Claude 3.5 Haiku",
                context_window=200_000,
                tier="fast",
                supports_tools=True,
            ),
            ModelInfo(
                name="claude-3-opus-latest",
                provider=ModelProvider.ANTHROPIC,
                display_name="Claude 3 Opus",
                context_window=200_000,
                tier="powerful",
                supports_tools=True,
            ),
        ]

    async def invoke(self, request: ModelRequest) -> ModelResponse:
        """调用 Anthropic 模型"""
        self.validate_request(request)

        headers = self._build_headers(request)
        payload = self._build_payload(request)

        async with self._client.stream(
            "POST",
            f"{self._base_url}/v1/messages",
            headers=headers,
            json=payload,
        ) as response:
            if response.status_code != 200:
                await self._handle_error(response)
            return self.parse_response(await response.json())

    async def invoke_stream(
        self,
        request: ModelRequest,
    ) -> AsyncIterator[ModelStreamEvent]:
        """流式调用 Anthropic 模型"""
        self.validate_request(request)
        request.stream = True

        headers = self._build_headers(request)
        payload = self._build_payload(request)

        async with self._client.stream(
            "POST",
            f"{self._base_url}/v1/messages",
            headers=headers,
            json=payload,
        ) as response:
            if response.status_code != 200:
                await self._handle_error(response)

            async for line in response.content:
                if line:
                    event = self._parse_stream_event(line)
                    if event:
                        yield event

    def validate_request(self, request: ModelRequest) -> None:
        """验证请求参数"""
        if not request.model:
            raise ValueError("Model name is required")
        if not request.messages:
            raise ValueError("Messages cannot be empty")
        if request.temperature < 0 or request.temperature > 2:
            raise ValueError("Temperature must be between 0 and 2")
        if request.max_tokens is not None and request.max_tokens <= 0:
            raise ValueError("max_tokens must be positive")

    def _build_headers(self, request: ModelRequest) -> dict[str, str]:
        """构建请求头"""
        return {
            "x-api-key": self._api_key,
            "anthropic-version": "2023-06-01",
            "content-type": "application/json",
            "anthropic-dangerous-direct-browser-access": "true",
        }

    def _build_payload(self, request: ModelRequest) -> dict[str, Any]:
        """构建请求载荷"""
        payload: dict[str, Any] = {
            "model": request.model,
            "messages": [self._message_to_anthropic(m) for m in request.messages],
            "temperature": request.temperature,
        }

        if request.system_prompt:
            payload["system"] = request.system_prompt
        if request.max_tokens:
            payload["max_tokens"] = request.max_tokens
        if request.top_p:
            payload["top_p"] = request.top_p
        if request.stop_sequences:
            payload["stop_sequences"] = request.stop_sequences
        if request.tools:
            payload["tools"] = request.tools

        return payload

    def _message_to_anthropic(self, message: Message) -> dict[str, Any]:
        """转换消息格式"""
        result: dict[str, Any] = {
            "role": message.role.value,
            "content": message.content,
        }
        if message.name:
            result["name"] = message.name
        if message.tool_calls:
            result["content"] = [
                {"type": "text", "text": message.content},
                *[{
                    "type": "tool_use",
                    "id": tc.id,
                    "name": tc.name,
                    "input": tc.arguments,
                } for tc in message.tool_calls],
            ]
        if message.tool_call_id:
            result["content"] = [
                {"type": "tool_result", "tool_use_id": message.tool_call_id,
                 "content": message.content},
            ]
        return result

    def parse_response(self, raw_response: Any) -> ModelResponse:
        """解析 Anthropic 响应"""
        content_blocks = raw_response.get("content", [])
        main_content = ""
        tool_calls: list[ToolCall] = []

        for block in content_blocks:
            if block.get("type") == "text":
                main_content += block.get("text", "")
            elif block.get("type") == "tool_use":
                tool_calls.append(ToolCall(
                    id=block.get("id", ""),
                    name=block.get("name", ""),
                    arguments=block.get("input", {}),
                ))

        return ModelResponse(
            content=main_content,
            model=raw_response.get("model", ""),
            stop_reason=raw_response.get("stop_reason"),
            tool_calls=tool_calls if tool_calls else None,
            usage=ModelUsage(
                input_tokens=raw_response.get("usage", {}).get("input_tokens", 0),
                output_tokens=raw_response.get("usage", {}).get("output_tokens", 0),
                total_tokens=sum([
                    raw_response.get("usage", {}).get("input_tokens", 0),
                    raw_response.get("usage", {}).get("output_tokens", 0),
                ]),
            ),
        )
```

```python
class OpenAIAdapter(ModelAdapter):
    """OpenAI 模型适配器"""

    def __init__(
        self,

        self._organization = organization
        self._timeout = timeout
        self._max_retries = max_retries
        self._client = httpx.AsyncClient(timeout=timeout)

    @property
    def provider(self) -> ModelProvider:
        return ModelProvider.OPENAI

    @property
    def supported_models(self) -> list[ModelInfo]:
        return [
            ModelInfo(
                name="gpt-4o",
                provider=ModelProvider.OPENAI,
                display_name="GPT-4o",
                context_window=128_000,
                tier="balanced",
                supports_tools=True,
                supports_vision=True,
            ),
            ModelInfo(

                provider=ModelProvider.OPENAI,
                display_name="GPT-4o Mini",
                context_window=128_000,
                tier="fast",
                supports_tools=True,
            ),
            ModelInfo(
                name="gpt-4-turbo",
                provider=ModelProvider.OPENAI,
                display_name="GPT-4 Turbo",
                context_window=128_000,
                tier="powerful",
                supports_tools=True,
            ),
        ]

    async def invoke(self, request: ModelRequest) -> ModelResponse:
        """调用 OpenAI 模型"""
        self.validate_request(request)

        headers = self._build_headers(request)
        payload = self._build_payload(request)

        response = await self._client.post(
            f"{self._base_url}/chat/completions",
            headers=headers,
            json=payload,
        )

        if response.status_code != 200:
            await self._handle_error(response)

        return self.parse_response(response.json())

    async def invoke_stream(
        self,
        request: ModelRequest,
    ) -> AsyncIterator[ModelStreamEvent]:
        """流式调用 OpenAI 模型"""
        self.validate_request(request)
        request.stream = True

        headers = self._build_headers(request)
        payload = self._build_payload(request)

        async with self._client.stream(
            "POST",
            f"{self._base_url}/chat/completions",
            headers=headers,
            json=payload,
        ) as response:
            if response.status_code != 200:
                await self._handle_error(response)

            async for line in response.content:
                if line:
                    event = self._parse_stream_event(line)
                    if event:
                        yield event

    def validate_request(self, request: ModelRequest) -> None:
        """验证请求参数"""
        if not request.model:
            raise ValueError("Model name is required")
        if not request.messages:
            raise ValueError("Messages cannot be empty")
        if request.temperature < 0 or request.temperature > 2:
            raise ValueError("Temperature must be between 0 and 2")

    def _build_headers(self, request: ModelRequest) -> dict[str, str]:
        """构建请求头"""
        headers = {
            "Authorization": f"Bearer {self._api_key}",
            "content-type": "application/json",
        }
        if self._organization:
            headers["OpenAI-Organization"] = self._organization
        return headers

    def _build_payload(self, request: ModelRequest) -> dict[str, Any]:
        """构建请求载荷"""
        payload: dict[str, Any] = {
            "model": request.model,
            "messages": [self._message_to_openai(m) for m in request.messages],
            "temperature": request.temperature,
            "stream": request.stream,
        }

        if request.max_tokens:
            payload["max_tokens"] = request.max_tokens
        if request.top_p:
            payload["top_p"] = request.top_p
        if request.stop_sequences:
            payload["stop"] = request.stop_sequences
        if request.tools:
            payload["tools"] = request.tools

        return payload

    def _message_to_openai(self, message: Message) -> dict[str, Any]:
        """转换消息格式"""
        result: dict[str, Any] = {
            "role": message.role.value,
            "content": message.content,
        }
        if message.name:
            result["name"] = message.name
        if message.tool_calls:
            result["tool_calls"] = [
                {
                    "id": tc.id,
                    "type": "function",
                    "function": {
                        "name": tc.name,
                        "arguments": json.dumps(tc.arguments),
                    },
                }
                for tc in message.tool_calls
            ]
        if message.tool_call_id:
            result["tool_call_id"] = message.tool_call_id
        return result

    def parse_response(self, raw_response: Any) -> ModelResponse:
        """解析 OpenAI 响应"""
        choices = raw_response.get("choices", [])
        if not choices:
            raise ValueError("No choices in response")

        choice = choices[0]
        message = choice.get("message", {})

        tool_calls: list[ToolCall] = []
        for tc in message.get("tool_calls", []):
            func = tc.get("function", {})
            tool_calls.append(ToolCall(
                id=tc.get("id", ""),
                name=func.get("name", ""),
                arguments=json.loads(func.get("arguments", "{}")),
            ))

        return ModelResponse(
            content=message.get("content", ""),
            model=raw_response.get("model", ""),
            stop_reason=choice.get("finish_reason"),
            tool_calls=tool_calls if tool_calls else None,
            usage=ModelUsage(
                input_tokens=raw_response.get("usage", {}).get("prompt_tokens", 0),
                output_tokens=raw_response.get("usage", {}).get("completion_tokens", 0),
                total_tokens=raw_response.get("usage", {}).get("total_tokens", 0),
            ),
        )
```

### 5.5 ModelRegistry 接口

```python
class ModelRegistry:
    """模型注册表"""

    def __init__(self) -> None:
        self._adapters: dict[ModelProvider, ModelAdapter] = {}
        self._models: dict[str, ModelInfo] = {}

    def register_adapter(self, adapter: ModelAdapter) -> None:
        """注册模型适配器"""
        self._adapters[adapter.provider] = adapter
        for model_info in adapter.supported_models:
            self._models[model_info.name] = model_info

    def get_adapter(self, provider: ModelProvider) -> ModelAdapter | None:
        """获取指定提供商的适配器"""
        return self._adapters.get(provider)

    def get_adapter_by_model(self, model_name: str) -> ModelAdapter | None:
        """根据模型名获取适配器"""
        model_info = self._models.get(model_name)
        if model_info:
            return self._adapters.get(model_info.provider)
        return None

    def get_model_info(self, model_name: str) -> ModelInfo | None:
        """获取模型信息"""
        return self._models.get(model_name)

    def list_models(
        self,
        provider: ModelProvider | None = None,
        tier: str | None = None,
    ) -> list[ModelInfo]:
        """列出模型"""
        models = list(self._models.values())
        if provider:
            models = [m for m in models if m.provider == provider]
        if tier:
            models = [m for m in models if m.tier == tier]
        return models

    def get_by_tier(self, tier: str) -> list[str]:
        """获取指定层级的模型名列表"""
        return [
            m.name for m in self._models.values()
            if m.tier == tier
        ]
```

## 6. 数据流

### 6.1 模型调用数据流

```
Orchestrator 路由决策
    │
    ▼
Agent 选择（基于复杂度）
    │
    ▼
Agent 持有特定 Model（由 Agent 决定用哪个模型）
    │
    ▼
PromptTemplateManager.render(template_name, variables)
    │
    ▼
ModelAdapter.invoke(request)
    │
    ├──► API 调用
    │
    ├──► 响应解析
    │
    └──► 错误处理 + 重试
    │
    ▼
ModelResponse 返回
    │
    ▼
Agent 处理响应（文本 / 工具调用）
```

### 6.2 上下文交互

| 层级 | 与 Model 的交互 |
| ---- | -------------- |
| Orchestrator | 路由决策，选择 Agent |
| Agent | 持有 Model 实例，决定使用哪个模型，处理响应 |
| Context | 提供 Prompt 变量（user_id, session_id, tools 等） |
| Tools | 提供可用工具列表，接收工具调用请求 |
| Session | 记录模型调用历史（用于成本统计） |
| EventBus | 发布 model_invoked / model_error 事件 |

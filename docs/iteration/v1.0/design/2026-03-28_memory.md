## 1. 模块概述

Memory 模块实现三层记忆系统，分别针对不同的记忆需求和访问模式。Working Memory 提供会话内的滑动窗口，Short-term Memory 提供向量语义搜索，Long-term Memory 提供持久化的用户偏好和经验。

### 1.1 三层记忆架构

| 层级 | 存储 | 生命周期 | 容量 | 用途 |
|------|------|----------|------|------|
| Working | 内存 | 会话内 | 固定窗口 | LLM Context 滑动窗口 |
| Short-term | Qdrant | 180 天 | 无限 | 语义搜索、RAG |
| Long-term | SQLite | 永久 | 无限 | 用户偏好、历史经验 |

### 1.2 差异化特性

| 特性 | 说明 |
|------|------|
| 三层分离 | 每层职责明确，互不干扰 |
| 自动压缩 | Working Memory 超限自动压缩 |
| RAG 支持 | Short-term 支持语义检索 |
| 偏好学习 | Long-term 学习用户偏好 |

## 3. MemoryManager

### 3.1 核心接口

```python
class MemoryManager:
    """三层记忆管理器"""

    def __init__(self, config: MemoryConfig):
        self.config = config
        self.working = WorkingMemory(config.working)
        self.short_term = ShortTermMemory(config.short_term)
        self.long_term = LongTermMemory(config.long_term)

    async def get_context(self, task: Task) -> MemoryContext:
        """
        获取任务上下文
        组合三层记忆
        """
        working = await self.working.get(task.session_id)
        short_term = await self.short_term.search(task.query, limit=5)
        long_term = await self.long_term.get_preferences(task.session_id)

        return MemoryContext(
            working=working,
            short_term=short_term,
            long_term=long_term
        )

    async def store(self, session_id: str, entry: MemoryEntry) -> None:
        """存储记忆条目"""
        # 根据类型存储到不同层
        if entry.type == MemoryType.WORKING:
            await self.working.append(session_id, entry)
        elif entry.type == MemoryType.SHORT_TERM:
            await self.short_term.store(entry)
        else:
            await self.long_term.store(entry)

    async def compress(self, session_id: str) -> None:
        """压缩 Working Memory"""
        if await self.working.size(session_id) > self.config.working.max_size:
            await self.working.compress(session_id)
```

## 4. Working Memory (工作记忆)

### 4.1 概述

Working Memory 是会话内的内存存储，使用滑动窗口机制管理 LLM Context 长度。

### 4.2 滑动窗口

```python
class WorkingMemory:
    """工作记忆 - 滑动窗口实现"""

    def __init__(self, config: WorkingMemoryConfig):
        self.max_size = config.max_size  # Token 数量
        self.window_type = config.window_type  # "sliding" | "summarize"

    async def get(self, session_id: str) -> List[Message]:
        """获取当前窗口内容"""
        return self.windows.get(session_id, [])

    async def append(self, session_id: str, entry: MemoryEntry) -> None:
        """追加记忆"""
        if session_id not in self.windows:
            self.windows[session_id] = []

        self.windows[session_id].append(entry)

        # 检查是否超限
        if self.token_count(session_id) > self.max_size:
            await self._shift_window(session_id)

    async def _shift_window(self, session_id: str) -> None:
        """滑动窗口：移除最旧的条目"""
        while self.token_count(session_id) > self.max_size * 0.8:
            self.windows[session_id].pop(0)

    async def compress(self, session_id: str) -> None:
        """压缩记忆（摘要）"""
        messages = self.windows.get(session_id, [])
        if not messages:
            return

        # 生成摘要
        summary = await self._summarize(messages[-10:])  # 最近 10 条

        # 保留摘要和最新消息
        self.windows[session_id] = [summary] + messages[-5:]
```

```python
    window_type: str = "sliding"  # sliding | summarize
    compression_threshold: float = 0.8  # 触发压缩的阈值
```

## 5. Short-term Memory (短时记忆)

### 5.1 概述

Short-term Memory 使用 Qdrant 向量数据库存储，支持语义检索和 RAG。

### 5.2 向量存储

```python
class ShortTermMemory:
    """短时记忆 - Qdrant 向量存储"""

    def __init__(self, config: ShortTermMemoryConfig):
        self.config = config
        self.client = QdrantClient(url=config.url)
        self.collection_name = config.collection_name

    async def store(self, entry: MemoryEntry) -> None:
        """存储记忆条目"""
        embedding = await self._embed(entry.content)

        point = PointStruct(
            id=str(entry.id),
            vector=embedding,
            payload={
                "content": entry.content,
                "session_id": entry.session_id,
                "created_at": entry.created_at.isoformat(),
                "type": entry.type.value,
            }
        )

        await self.client.upsert(
            collection_name=self.collection_name,
            points=[point]
        )

    async def search(
        self,
        query: str,
        limit: int = 5,
        session_id: Optional[str] = None,
    ) -> List[MemoryEntry]:
        """语义检索"""
        query_embedding = await self._embed(query)

        filter_condition = None
        if session_id:
            filter_condition = Filter(
                must=[
                    FieldCondition(
                        key="session_id",
                        match=MatchValue(value=session_id)
                    )
                ]
            )

        results = await self.client.search(
            collection_name=self.collection_name,
            query_vector=query_embedding,
            limit=limit,
            query_filter=filter_condition,
        )

        return [
            MemoryEntry(
                id=int(hit.id),
                content=hit.payload["content"],
                session_id=hit.payload["session_id"],
                type=MemoryType.SHORT_TERM,
                created_at=datetime.fromisoformat(hit.payload["created_at"]),
            )
            for hit in results
        ]

    async def delete_expired(self, ttl_days: int = 180) -> int:
        """删除过期记忆"""
        cutoff = datetime.now() - timedelta(days=ttl_days)

        # 获取过期条目
        results = await self.client.scroll(
            collection_name=self.collection_name,
            scroll_filter=Filter(
                must=[
                    FieldCondition(
                        key="created_at",
                        range=Range(lt=cutoff.isoformat())
                    )
                ]
            )
        )

        # 删除
        ids_to_delete = [int(hit.id) for hit in results.points]
        if ids_to_delete:
            await self.client.delete(
                collection_name=self.collection_name,
                points_selector=PointIdsList(points=ids_to_delete)
            )

        return len(ids_to_delete)
```

## 6. Long-term Memory (长时记忆)

### 6.1 概述

Long-term Memory 使用 SQLite 持久化存储，保存用户偏好和历史经验。

### 6.2 用户偏好

```python
class LongTermMemory:
    """长时记忆 - SQLite 持久化"""

    def __init__(self, config: LongTermMemoryConfig):
        self.config = config
        self.db_path = config.db_path
        self._init_db()

    def _init_db(self) -> None:
        """初始化数据库"""
        conn = sqlite3.connect(self.db_path)
        conn.execute("""
            CREATE TABLE IF NOT EXISTS preferences (
                key TEXT PRIMARY KEY,
                value TEXT NOT NULL,
                updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
            )
        """)
        conn.execute("""
            CREATE TABLE IF NOT EXISTS experiences (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                session_id TEXT,
                task_type TEXT,
                success BOOLEAN,
                summary TEXT,
                details JSON,
                created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
            )
        """)
        conn.close()

    async def get_preferences(self, session_id: str) -> UserPreferences:
        """获取用户偏好"""
        conn = sqlite3.connect(self.db_path)
        cursor = conn.execute(
            "SELECT key, value FROM preferences WHERE key LIKE ?",
            (f"{session_id}:%",)
        )
        prefs = {row[0]: row[1] for row in cursor}
        conn.close()

        return UserPreferences(
            preferred_model=prefs.get(f"{session_id}:model"),
            preferred_language=prefs.get(f"{session_id}:language", "en"),
            coding_style=prefs.get(f"{session_id}:coding_style"),
        )

    async def update_preference(self, session_id: str, key: str, value: str) -> None:
        """更新偏好"""
        conn = sqlite3.connect(self.db_path)
        conn.execute(
            "INSERT OR REPLACE INTO preferences (key, value) VALUES (?, ?)",
            (f"{session_id}:{key}", value)
        )
        conn.commit()
        conn.close()

    async def store_experience(self, experience: Experience) -> None:
        """存储经验"""
        conn = sqlite3.connect(self.db_path)
        conn.execute(
            """
            INSERT INTO experiences (session_id, task_type, success, summary, details)
            VALUES (?, ?, ?, ?, ?)
            """,
            (
                experience.session_id,
                experience.task_type,
                experience.success,
                experience.summary,
                json.dumps(experience.details),
            )
        )
        conn.commit()
        conn.close()

    async def get_similar_experiences(
        self,
        task_type: str,
        limit: int = 5,
    ) -> List[Experience]:
        """获取相似经验"""
        conn = sqlite3.connect(self.db_path)
        cursor = conn.execute(
            """
            SELECT * FROM experiences
            WHERE task_type = ? AND success = 1
            ORDER BY created_at DESC
            LIMIT ?
            """,
            (task_type, limit)
        )
        rows = cursor.fetchall()
        conn.close()

        return [
            Experience(
                id=row[0],
                session_id=row[1],
                task_type=row[2],
                success=bool(row[3]),
                summary=row[4],
                details=json.loads(row[5]),
                created_at=datetime.fromisoformat(row[6]),
            )
            for row in rows
        ]
```

```
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│                    MemoryManager                             │
│                                                             │
│  ┌─────────────────┐                                        │
│  │   Working       │ ──── 当前会话上下文                     │

│  └────────┬────────┘                                        │
│           │                                                  │
│  ┌────────▼────────┐                                        │
│  │   Short-term    │ ──── RAG 语义检索                       │
│  │   Memory         │ ──── 相关历史记忆                       │
│  └────────┬────────┘                                        │
│           │                                                  │
│  ┌────────▼────────┐                                        │
│  │   Long-term     │ ──── 用户偏好                           │

│  └────────┬────────┘                                        │
│           │                                                  │
│           ▼                                                  │
│  ┌─────────────────────────────────────────┐                │

│  └─────────────────────────────────────────┘                │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
返回 MemoryContext
```

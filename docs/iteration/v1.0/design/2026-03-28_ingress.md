| 状态 | 规划中 |

## 1. 组件列表

| 组件 | 路径 | 职责 |
|------|------|------|
| CLI (Typer) | `mozi/ingress/cli/` | 命令行交互 |
| Web UI (FastAPI) | `mozi/ingress/web/` | REST API、WebSocket |
| IDE Extension | `mozi/ingress/ide/` | VSCode/JetBrains 集成 |

## 2. CLI 命令

| 命令 | 说明 |
|------|------|
| `mozi` | 启动交互会话 |
| `mozi "task"` | 带初始提示的会话 |
| `mozi -p "query"` | 非交互模式 |
| `mozi -c` | 继续最近会话 |
| `mozi -r <session>` | 恢复指定会话 |

## 3. 依赖关系

- **依赖模块**: 无
- **被依赖模块**: Orchestration Layer

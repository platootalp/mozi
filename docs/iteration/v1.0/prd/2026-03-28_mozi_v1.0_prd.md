# Mozi v1.0 PRD

## 1. 概述与目标

Mozi AI Coding Agent MVP 版本，交付一个功能完整的 AI 编程助手，具备：
- 基于复杂度的智能任务路由
- 安全的工具执行框架
- 多Agent协作能力
- 持久化记忆系统
- MCP 协议集成

## 2. 功能优先级

### P0 - 核心功能（必须交付）

| ID | 功能 | 描述 |
|----|------|------|
| P0.1 | CLI 入口 | 完整的命令行界面 |
| P0.2 | FastPath | SIMPLE 任务直接执行（≤40 复杂度） |
| P0.3 | 内置工具 | read/write/edit/bash/grep/glob |
| P0.4 | 权限系统 | deny-first 五级权限 |
| P0.5 | 审计日志 | 所有敏感操作记录 |
| P0.6 | 复杂度评估 | 四维评分引擎 |
| P0.7 | MEDIUM 路由 | 41-70 复杂度任务处理 |
| P0.8 | 会话管理 | SQLite 持久化 |

### P1 - 增强功能（应该交付）

| ID | 功能 | 描述 |
|----|------|------|
| P1.1 | DAG 规划 | COMPLEX 任务（>70）分解 |
| P1.2 | 多Agent | Agent 池与协调 |
| P1.3 | 检查点恢复 | 任务状态保存与恢复 |
| P1.4 | 三层记忆 | Working/Short-term/Long-term |
| P1.5 | 向量检索 | Qdrant 集成 |
| P1.6 | MCP 客户端 | 标准传输支持 |
| P1.7 | HITL | 敏感操作人工审批 |

### P2 - 高级功能（可选）

| ID | 功能 | 描述 |
|----|------|------|
| P2.1 | Web UI | FastAPI 管理界面 |
| P2.2 | WebSocket | 实时流式输出 |
